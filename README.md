## üõ†Ô∏è Widget yang Digunakan

| Widget                                        | Fungsi                                                                                                                                                                                                                                       |
| --------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **MaterialApp**                               | Widget utama yang membungkus seluruh aplikasi Flutter dengan dukungan Material Design, mengatur theme, routing, dan home screen. Digunakan di `main.dart` untuk menginisialisasi app dengan Material 3 dan color scheme berbasis seed color. |
| **StatelessWidget**                           | Widget dasar yang tidak memiliki state internal yang berubah. Digunakan sebagai base class untuk `MyShopMiniApp` dan `ProductListScreen`.                                                                                                    |
| **StatefulWidget**                            | Widget yang memiliki state yang dapat berubah. Digunakan di `HomeScreen`, `CategoryCard`, `ProductCard`, dan `ProductDetailScreen` untuk menangani interaksi user dan animasi.                                                               |
| **Scaffold**                                  | Menyediakan struktur dasar untuk layar visual seperti AppBar, body, dan background. Digunakan di semua screen untuk membangun layout halaman dengan konsistensi.                                                                             |
| **AppBar**                                    | Bar navigasi atas yang menampilkan title, leading (back button), dan actions. Digunakan di `ProductListScreen` dan `ProductDetailScreen` dengan custom styling sesuai kategori.                                                              |
| **SafeArea**                                  | Memastikan konten tidak tertutup oleh notch, status bar, atau area sistem lainnya. Digunakan di `HomeScreen` untuk menjaga konten tetap dalam area aman.                                                                                     |
| **Container**                                 | Widget serbaguna untuk styling, positioning, dan sizing. Digunakan untuk membuat card wrapper, icon container, badge, dan background dengan decoration, padding, margin, dan transform.                                                      |
| **Column**                                    | Mengatur child widgets secara vertikal (atas ke bawah). Digunakan di semua screen untuk layout vertikal seperti header, content, dan list items.                                                                                             |
| **Row**                                       | Mengatur child widgets secara horizontal (kiri ke kanan). Digunakan di `CategoryCard` untuk menampilkan icon, text, dan arrow secara horizontal.                                                                                             |
| **Padding**                                   | Menambahkan ruang kosong (padding) di sekitar child widget. Digunakan secara ekstensif untuk spacing internal dalam card, button, dan container.                                                                                             |
| **Expanded**                                  | Membuat child widget mengisi ruang yang tersedia dalam flex container seperti Row atau Column. Digunakan di `CategoryCard` untuk membuat text mengisi ruang yang tersedia, dan di `ProductListScreen` untuk GridView.                        |
| **ListView.builder**                          | Membuat list scrollable secara dinamis berdasarkan item count. Digunakan di `HomeScreen` untuk menampilkan daftar kategori secara vertikal dengan animasi masuk.                                                                             |
| **GridView.builder**                          | Membuat grid scrollable secara dinamis berdasarkan item count. Digunakan di `ProductListScreen` untuk menampilkan daftar produk dalam grid 2 kolom dengan spacing yang konsisten.                                                            |
| **SliverGridDelegateWithFixedCrossAxisCount** | Delegate untuk GridView yang menentukan jumlah kolom tetap (2 kolom), aspect ratio (0.8), dan spacing (20px). Digunakan di `ProductListScreen` untuk konfigurasi layout grid produk.                                                         |
| **GestureDetector**                           | Mendeteksi berbagai gesture seperti tap, long press, dan drag. Digunakan di `CategoryCard` dan `ProductCard` untuk menangani interaksi tap dengan state management (pressed/hover effect).                                                   |
| **InkWell**                                   | Alternatif GestureDetector dengan efek ripple Material Design. (Meskipun dalam kode ini lebih banyak menggunakan GestureDetector untuk custom animation).                                                                                    |
| **Icon**                                      | Menampilkan ikon dari Material Icons. Digunakan untuk ikon kategori, produk, navigation arrow, dan back button dengan ukuran dan warna yang dapat dikustomisasi.                                                                             |
| **Text**                                      | Menampilkan teks dengan styling seperti font size, weight, color, dan alignment. Digunakan di seluruh app untuk title, subtitle, nama kategori/produk, harga, dan label.                                                                     |
| **SizedBox**                                  | Widget kosong dengan ukuran tetap untuk spacing antar elemen. Digunakan di semua screen untuk jarak vertikal dan horizontal yang konsisten.                                                                                                  |
| **Center**                                    | Memusatkan child widget secara horizontal dan vertikal. Digunakan di `ProductDetailScreen` untuk memusatkan konten detail produk.                                                                                                            |
| **AnimatedContainer**                         | Container yang dapat berubah propertinya dengan animasi smooth. Digunakan di `CategoryCard` dan `ProductCard` untuk efek scale saat ditekan dan hover effect dengan transform.                                                               |
| **TweenAnimationBuilder**                     | Builder untuk membuat animasi custom dengan tween (interpolasi nilai). Digunakan di `HomeScreen` untuk animasi masuk kategori dengan translate dan opacity, serta di `ProductCard` untuk scale dan fade-in effect.                           |
| **AnimationController**                       | Controller untuk mengontrol animasi secara manual. Digunakan di `HomeScreen` dan `ProductDetailScreen` dengan `SingleTickerProviderStateMixin` untuk animasi fade-in saat screen dibuka.                                                     |
| **FadeTransition**                            | Widget untuk animasi fade (opacity). Digunakan di `HomeScreen` dan `ProductDetailScreen` untuk efek muncul secara perlahan saat screen pertama kali dimuat.                                                                                  |
| **ScaleTransition**                           | Widget untuk animasi scale (perbesaran/pengecilan). Digunakan di `ProductDetailScreen` untuk animasi icon produk dengan efek elastic bounce.                                                                                                 |
| **Transform.scale**                           | Mengubah skala widget tanpa mempengaruhi layout. Digunakan di `ProductCard` untuk efek zoom-in saat animasi masuk.                                                                                                                           |
| **Transform.translate**                       | Mengubah posisi widget tanpa mempengaruhi layout. Digunakan di `HomeScreen` untuk animasi slide-up categories dan di `ProductCard` untuk hover effect (naik sedikit).                                                                        |
| **BoxDecoration**                             | Dekorasi untuk Container dengan border, borderRadius, color, gradient, dan boxShadow. Digunakan secara ekstensif untuk styling card, button, dan background dengan shadow effect.                                                            |
| **BorderRadius.circular**                     | Membuat sudut melengkung dengan radius tertentu. Digunakan di semua card dan container untuk tampilan modern dan soft.                                                                                                                       |
| **BoxShadow**                                 | Menambahkan bayangan pada Container. Digunakan dengan opacity dan blur radius yang bervariasi untuk depth effect sesuai warna kategori.                                                                                                      |
| **RadialGradient**                            | Gradient melingkar dari center point. Digunakan di `HomeScreen` sebagai background dengan gradient cyan ke putih untuk efek visual yang menarik.                                                                                             |
| **Color**                                     | Mendefinisikan warna dengan berbagai format (hex, ARGB). Digunakan untuk mendefinisikan color palette aplikasi dan warna per kategori.                                                                                                       |
| **Navigator.push**                            | Navigasi ke screen baru dengan menambahkan route ke stack. Digunakan untuk berpindah dari `HomeScreen` ke `ProductListScreen`, dan dari `ProductListScreen` ke `ProductDetailScreen`.                                                        |
| **Navigator.pop**                             | Kembali ke screen sebelumnya dengan menghapus route teratas dari stack. Digunakan di back button `ProductListScreen` dan `ProductDetailScreen`.                                                                                              |
| **MaterialPageRoute**                         | Route dengan transisi Material Design. Digunakan untuk semua navigasi antar screen dengan transisi slide dari kanan.                                                                                                                         |
| **IconButton**                                | Button dengan icon sebagai child. Digunakan untuk back button di AppBar dengan custom color sesuai kategori.                                                                                                                                 |
| **Opacity**                                   | Mengatur transparansi widget. Digunakan dalam animasi fade-in di `ProductCard` dan kategori.                                                                                                                                                 |
| **SingleTickerProviderStateMixin**            | Mixin untuk menyediakan Ticker untuk AnimationController. Digunakan di `HomeScreen` dan `ProductDetailScreen` agar animasi berjalan dengan frame rate yang stabil.                                                                           |
| **TickerProviderStateMixin**                  | Versi mixin yang mendukung multiple AnimationController. Digunakan di `HomeScreen` untuk kemungkinan multiple animations.                                                                                                                    |
